# Calculadora de Juros com JavaScript 
A calculadora foi desenvolvida como aplicação em terminal.

Para isso foi necessário:
1. Criar pasta para projeto
2. Instalar na pasta do projeto, o plugin readline-sync (para abrir o terminal e executar a calculadora)
3. No terminal digite npm init -y  e após npm install realine-sync
4. No arquivo package.json após description adicione "type":"module";
5. Crie um apelido para o plugin e importe no início do arquivo js.
6. Para executar a calculadora: no terminal digite node index.js

## Regra de negócio
Para quantidade de dias de vencimento >= 15 a taxa de juros é 10%
e quantidade de dias de vencimento < 15 a taxa é de 5%
